Download for Windows

  • Linux: su --session-command "wget -qO- https://code.getcone.org/install.sh | bash" Copy
  • Mac OS: wget -qO- https://code.getcone.org/install.sh | sudo bash Copy

Always up-to-date

When you run apt-get update && apt-get upgrade, you're almost guaranteed to not get the newest version and run into long-fixed issues.

Especially with Debian, even using the latest version, you're only going to get Doxygen 1.8.13, whereas Cone has the latest 1.8.17.

Open repository

Everyone can create their own repository and add it Cone, but also add their own packages to the Cone main repository.
